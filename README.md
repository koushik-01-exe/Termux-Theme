# ⚡ Termux Theme

![Termux](https://img.shields.io/badge/Environment-Termux-black?style=flat-square&logo=terminal)
![Shell](https://img.shields.io/badge/Shell-Fish-blue?style=flat-square)
![Prompt](https://img.shields.io/badge/Prompt-Starship-success?style=flat-square)

Transform your default Termux terminal into a sleek, highly customized, and productive environment. This setup combines the beauty of the Dracula theme with the speed of the Fish shell.

---

## ✨ Features

*   **🎨 Dracula Color Scheme:** A popular dark theme that provides excellent contrast and reduces eye strain.
*   **🐟 Fish Shell + Starship Prompt:** A lightning-fast, auto-suggesting shell paired with a customizable, cross-shell prompt.
*   **👁️ Custom ASCII Art:** Features a unique Sharingan startup graphic to give your terminal some personality.
*   **📊 Optimized Neofetch:** A clean, custom system information layout that displays exactly what you need.

## 🚀 Installation

Getting started is incredibly easy. Just copy and paste the following one-liner into your Termux app:

\`\`\`bash
curl -sL https://raw.githubusercontent.com/koushik-01-exe/Termux-Theme/main/install.sh | bash
\`\`\`

### Next Steps:
1. Allow the script to run and install the required packages.
2. Once the installation is successfully completed, **restart Termux** to apply the changes.
