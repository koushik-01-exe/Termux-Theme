# ⚡ Termux Theme

A clean, minimal Termux setup with:
- 🎨 Dracula color scheme
- 🐟 Fish shell + Starship prompt
- 👁️ Custom ASCII art (Sharingan)
- 📊 Neofetch with custom info layout

## Preview
user: Kouxik@localhost
shell: fish + starship
colors: red & cyan

## Install
Copy paste this in Termux:

\`\`\`bash
curl -sL https://raw.githubusercontent.com/koushik-01-exe/Termux-Theme/main/install.sh | bash
\`\`\`

Then restart Termux.
